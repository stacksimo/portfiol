# Stacksimo — Portfolio

Professional freelance portfolio site for Stacksimo: site building, SaaS products, and landing pages.

Static site — no build step, no dependencies.

## Run locally

```bash
python3 -m http.server 8080
# open http://localhost:8080
```

## Deploy

Works out of the box on GitHub Pages, Netlify, Vercel, or Cloudflare Pages — just point the host at the repo root.

## Customize

- Content: edit `index.html` (work, services, about, contact email)
- Colors/typography: edit the `:root` variables in `styles.css`
